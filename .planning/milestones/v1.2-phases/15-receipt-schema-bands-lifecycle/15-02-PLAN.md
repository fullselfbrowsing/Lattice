# Plan 15-02: Tripwire band pipeline + lifecycle event vocabulary

**Status:** Complete (retro).
**Wave:** 1
**REQ-IDs:** BAND-01, BAND-02, BAND-03, BAND-04, BAND-05, LIFECYCLE-01
**Originating commit:** `ba6172c` feat(contract): add tripwire band pipeline primitive.

## Goal

Introduce a new `packages/lattice/src/contract/bands.ts` module exporting a `createHookPipeline()` factory with three priority bands (`SAFETY` > `OBSERVABILITY` > `EXTENSION`), per-handler matcher regex, race-with-log budget enforcement (default 100ms), frozen-context handler isolation, and irreversible mid-session registration freeze. Export a `HookLifecycleEvent` union vocabulary structurally separate from `RunEventKind`.

## Tasks

1. New `packages/lattice/src/contract/bands.ts` (+261 lines).
   - Export `BAND` const enum: `SAFETY=0`, `OBSERVABILITY=1`, `EXTENSION=2`.
   - Export `HookLifecycleEvent` union: `BEFORE_PROVIDER`, `AFTER_PROVIDER`, `BEFORE_TOOL`, `AFTER_TOOL`.
   - Export `RegisterOptions` type with optional `matcher: RegExp` and `budgetMs: number` (default 100).
   - Export `createHookPipeline(tracer?: TracerLike)` factory.
   - Pipeline `register(event, handler, options?)` appends to the in-band registration list. Throws `Error` with `name === "PIPELINE_FROZEN"` if `freeze()` already called.
   - Pipeline `run(event, context)` iterates bands in order (SAFETY → OBSERVABILITY → EXTENSION), within each band iterates registration order, applies matcher filter, and races each handler against `budgetMs`. Timeout emits `HOOK_TIMEOUT` via tracer (no abort).
   - Pipeline freezes handler contexts with `structuredClone` + `Object.freeze` before invocation.
   - Pipeline `freeze()` flips an internal latch; irreversible.
2. New `packages/lattice/src/contract/bands.test.ts` (+234 lines) covering: registration order within band; cross-band ordering; matcher regex filter; budget timeout emits `HOOK_TIMEOUT`; frozen-context immutability; irreversible freeze; lifecycle event union exhaustiveness.
