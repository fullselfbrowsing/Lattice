    - Use `readSseEvents(response)`.

    - For `content_block_delta` with `delta.type === "input_json_delta"`, append `partial_json` to the matching tool block.
    - On `content_block_stop`, parse completed tool input JSON and validate through `validateToolCallRequests` when configured.
    - Parse `message_delta.usage` or final usage fields into `usage` and `normalizedUsage`.
    - At stream end, yield `complete` with sanitized raw outputs, usage, normalized usage, validated tool calls if any, and content-bearing `rawResponse`.
    - Ignore unknown event types.
