  <action>
    Create `scripts/check-package-version-surfaces.mjs` as a zero-dependency Node 24 script using `node:child_process`, `node:fs/promises`, `node:path`, `node:os`, and `node:url`.

    Required behavior:
    1. Hard-code package entries:
       ```javascript
       const PACKAGES = [
         { dir: "packages/lattice", name: "@full-self-browsing/lattice", kind: "runtime" },
         { dir: "packages/lattice-cli", name: "@full-self-browsing/lattice-cli", kind: "cli" },
       ];
       ```
    2. For each package, create a temp directory, run `pnpm pack --pack-destination <tmp>` in the package directory, and extract the produced `.tgz` into a second temp directory with `tar -xzf`.
    3. Read `package/package.json` from the extracted tarball.
    4. For runtime, run:
       ```javascript
       node --input-type=module -e "const mod = await import(process.argv[1]); if (mod.latticeVersion !== process.argv[2]) { console.error(`${mod.latticeVersion} !== ${process.argv[2]}`); process.exit(1); }" <extracted>/package/dist/index.js <manifest.version>
       ```
       Use an equivalent `spawn` call; do not shell interpolate paths.
    5. For CLI, run `node <extracted>/package/dist/cli.js --help` with `NO_COLOR=1`, strip ANSI with `/\x1B\[[0-?]*[ -/]*[@-~]/g`, and assert the output contains `(lattice v${manifest.version})` and does not contain `v0.0.0`.
    6. Print `[check-package-version-surfaces] OK - inspected 2 tarballs` on success.
    7. On failure, print `[check-package-version-surfaces] FAIL - <package> <reason>` and exit 1.
    8. Always remove temp directories in `finally` blocks.

    Add this root package script to `package.json`:
    ```json
    "check:package-version": "node scripts/check-package-version-surfaces.mjs"
    ```
  </action>
  <verify>
    <automated>bash -c 'pnpm -r build && node scripts/check-package-version-surfaces.mjs'</automated>
  </verify>

    - `scripts/check-package-version-surfaces.mjs` contains `latticeVersion`.
