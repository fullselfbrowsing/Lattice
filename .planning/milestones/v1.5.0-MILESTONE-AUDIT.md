- `75806c4 fix(55): enforce eval facade boundary`
- Moved the eval kernel to `packages/lattice/src/eval/agent-run.ts`.
- Kept `packages/lattice/src/agent/eval.ts` as a compatibility re-export for existing source imports.
- Updated `packages/lattice/src/eval.ts` and root exports to use the neutral eval module.
- Added eval facade enforcement to `scripts/check-lattice-module-boundaries.mjs`.
