## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Provider response â†’ runtime tripwire evaluation | Provider output is untrusted. Tripwires are the gate; bypassing them = trust failure. |
| Fallback chain â†’ tripwire-violated error | Generic retry logic could mistake a violation for a transient error and burn budget on retries (Pitfall #4). |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-06 | Elevation of Privilege | Fallback chain retrying past a tripwire | mitigate | Early-return inside the violation branch terminates `runWithConfig` before the `for` loop advances.
