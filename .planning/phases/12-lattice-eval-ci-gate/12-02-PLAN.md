Per-fixture pipeline (mirrors Phase 11's 6-stage `repro.ts` pattern):
```
Stage 1 — walker emits WalkedEntry
   load-failed -> FixtureReport { verdict: "load-failed", regressionKind: null, usage: null, qualityScore: null }
Stage 2 — materializeReplayEnvelope(envelope, { artifactLoader, keySet })
   failure  -> verdict: "load-failed" (verify-failed / artifact-load-failed / envelope-malformed all collapse here)
Stage 3 — second verifyReceipt for typed body (microsecond cost; same trick as repro.ts)
Stage 4 — replayOffline(envelopeReplay)
   failure  -> verdict: "load-failed"
Stage 5 — Exact class: sha256(JSON.stringify(result.outputs)) vs body.outputHash
   mismatch -> verdict: "drift", regressionKind: "output-hash-mismatch"  (SHORT-CIRCUIT — skip judge)
   body.outputHash === null -> verdict: "load-failed" (failure receipt, no diff target)
   match    -> proceed
Stage 6 — Semantic-cheap class: NO-OP in v1.1 (CONTEXT.md "no-op in v1.1 unless --outputs flag")
   future hook: when args.schemaPath is set, run Standard Schema validation; for now, skip.
Stage 7 — Semantic-expensive (judge) class: ONLY when body.qualityFloor is non-null
   runJudgeWithN(judge, { fixtureId, output, modelFingerprint, prompt }, judgeN, cache)
   qualityScore = the median; cached flag recorded internally but not surfaced in the report.
Stage 8 — Baseline cost gate: compareCost(replayCostUsd, baselineEntry.usage.costUsd, costTolerance)
   regression -> regressionKind: "cost-regression"; verdict: "regression" (even if Exact matched).
Stage 9 — Baseline quality gate: compareQuality(qualityScore, baselineEntry.qualityFloor.score, qualityTolerance)
   regression -> regressionKind: "quality-regression"; verdict: "regression".
Stage 10 — Missing baseline entry: newFixture++, verdict: "match", regressionKind: null,
           deltaCostPct/deltaQuality both null (CONTEXT.md "record but don't flag").
```

    Implementation rules:

    1. Default deps: `judge = noopJudge`, `now = () => new Date().toISOString()`, `loadKeySet = loadKeySetFromPath`, `buildArtifactLoader = createFilesystemArtifactLoader`.
    2. Load keyset once at session start. On failure, propagate — the caller (Plan 03) maps to exit 2.
    3. Load baseline. If `config.initBaseline === true`, skip the load and use `baseline = undefined`. If load fails AND `initBaseline === false`, propagate the BaselineLoadError — Plan 03 maps to exit 2.
    4. Create one `judgeCache = createDiskJudgeCache(config.judgeCacheDir)` for the whole session.
    5. For each `entry` from `walkReceiptsDirectory(config.fixturesDir)`:
       a. If `isWalkedReceiptError(entry)`: push `{ fixtureId: entry.id, verdict: "load-failed", regressionKind: null, usage: null, qualityScore: null, deltaCostPct: null, deltaQuality: null }`. Continue.
       b. Otherwise: run the per-fixture pipeline below. Wrap each stage's failures so one bad fixture never aborts the whole session.

    Per-fixture pipeline:
    ```typescript
    let envelopeReplay;
    try {
      envelopeReplay = await materializeReplayEnvelope(entry.envelope, { artifactLoader, keySet });
    } catch { /* push verdict: "load-failed"; continue */ }

    const verifyResult = await verifyReceipt(entry.envelope, keySet);
    if (!verifyResult.ok) { /* push verdict: "load-failed"; continue */ }
    const body: CapabilityReceiptBody = verifyResult.body;

    const replay = await replayOffline(envelopeReplay);
    if (!replay.ok) { /* push verdict: "load-failed"; continue */ }

    // Exact class
    if (body.outputHash === null) { /* push verdict: "load-failed"; continue */ }
    const actualHash = await sha256Hex(JSON.stringify(replay.outputs));
    let regressionKind: RegressionKind = null;
    let verdict: FixtureReport["verdict"] = "match";
    if (actualHash !== body.outputHash) {
      regressionKind = "output-hash-mismatch";
      verdict = "drift";
      // SHORT-CIRCUIT: skip semantic-cheap + semantic-expensive
    } else {
      // Semantic-cheap class: NO-OP in v1.1 (CONTEXT.md). Future hook lives here.
    }

    // Semantic-expensive class: only when Exact passed AND body declares qualityFloor
    let qualityScore: number | null = null;
    if (verdict === "match" && body.qualityFloor !== undefined && body.qualityFloor !== null) {
      const judgeResult = await runJudgeWithN(
        deps.judge ?? noopJudge,
        {
          fixtureId: entry.id,
          output: replay.outputs,
          modelFingerprint: body.model.observed?.fingerprint ?? body.model.requested,
          prompt: config.judgePrompt,
        },
        config.judgeN,
        judgeCache,
      );
      qualityScore = judgeResult.score;
    }

    // Baseline gates — only when baseline is loaded AND fixture present
    let deltaCostPct: number | null = null;
    let deltaQuality: number | null = null;
    const baselineEntry: BaselineEntry | undefined = baseline?.fixtures[entry.id];
    if (baseline !== undefined && baselineEntry !== undefined) {
      // Cost gate (independent of Exact verdict — even matches can regress on cost)
      const costCmp = compareCost(body.usage.costUsd ?? "0", baselineEntry.usage.costUsd, config.costTolerance);
      deltaCostPct = costCmp.deltaPct;
      if (costCmp.regressed && verdict === "match") {
        verdict = "regression";
        regressionKind = "cost-regression";
      }

      // Quality gate (only when both sides recorded a score)
      if (qualityScore !== null && baselineEntry.qualityFloor !== null) {
        const qualCmp = compareQuality(qualityScore, baselineEntry.qualityFloor.score, config.qualityTolerance);
        deltaQuality = qualCmp.delta;
        if (qualCmp.regressed && verdict === "match") {
          verdict = "regression";
          regressionKind = "quality-regression";
        }
      }
    } else if (baseline !== undefined && baselineEntry === undefined) {
      // New fixture: record but don't flag
      newFixtures++;
    }

    fixtures.push({
      fixtureId: entry.id,
      verdict,
      regressionKind,
      usage: { costUsd: body.usage.costUsd ?? "0", promptTokens: body.usage.promptTokens, completionTokens: body.usage.completionTokens },
      qualityScore,
      deltaCostPct,
      deltaQuality,
    });
    ```
