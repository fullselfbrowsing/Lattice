# Plan 20-01: AgentHost interface + noop reference impl + recovery markers

**Status:** Not started
**Wave:** 1
**REQ-IDs:** HOST-01, HOST-02, TRACE-EXT-01

## Goal

Promote the Phase 19 `AgentHost` forward-declaration to a full interface with three optional pluggable seams (scheduler, transport, storage). Ship `createNoopAgentHost()` as the Node-test reference implementation. Extend `RunEventKind` with three new recovery markers (`recovery.start` / `recovery.complete` / `recovery.failed`).

## Tasks

### Task 1: `packages/lattice/src/agent/host.ts` (NEW)

Export:
- `AgentHost` interface — `{ kind: "agent-host"; scheduler?: AgentScheduler; transport?: AgentTransport; storage?: AgentStorage<TState> }`.
- `AgentScheduler` interface — `{ scheduleNext(iterationIndex: number): Promise<void> }`.
- `AgentTransport` interface — `{ call(provider: ProviderAdapter, request: ProviderRunRequest): Promise<ProviderRunResponse> }`.
- `AgentStorage<TState>` interface — `{ save(snapshot: SerializedSnapshot): Promise<void>; load(): Promise<SerializedSnapshot | null>; clear(): Promise<void> }`.
- `AgentSnapshot` interface — `{ version: "agent-snapshot/v1"; iterationIndex: number; conversation: readonly ConversationTurn[]; cumulativeUsage: Usage; providerName: string; capturedAt: string }`.
- `createNoopAgentHost()` factory returning an `AgentHost` with all three seams defaulting to no-op behavior.

### Task 2: `packages/lattice/src/tracing/tracing.ts`

Add three literals to `RunEventKind` (additive, at end of union):
- `"recovery.start"`
- `"recovery.complete"`
- `"recovery.failed"`

### Task 3: `packages/lattice/src/agent/types.ts`

Replace the Phase 19 forward-decl with a re-export from `host.js`:
```ts
export type { AgentHost, AgentScheduler, AgentTransport, AgentStorage, AgentSnapshot } from "./host.js";
```

Add new optional field on `AgentIntent`:
```ts
readonly survivabilityAdapter?: SurvivabilityAdapter<AgentSnapshot>;
```

### Task 4: Tests

`packages/lattice/src/agent/host.test.ts` covering:
- `createNoopAgentHost()` returns the expected shape (3 seams).
- scheduler.scheduleNext resolves immediately.
- transport.call delegates to provider.execute.
- storage.save / load / clear round-trip (load returns null, save+load round-trips, clear empties).
