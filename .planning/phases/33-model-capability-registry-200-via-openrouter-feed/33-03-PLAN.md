<objective>
Ship the build-time codegen pipeline that turns OpenRouter's live `/api/v1/models` feed into a typed inline TypeScript array. Three deliverables plus a vitest suite:

1. **`scripts/capabilities/classifier.mjs`** — pure Node ESM, zero external deps. Exports `classify(rawEntry) -> { originFamily, trainingClass, reasoningSurface, toolCallSurface, knownFailureModes, recommendedPromptStrategy }` using the hybrid D-01 strategy: provider-prefix heuristic first, ~20-entry family-substring override table second, permissive `open_weight_instruct` + stderr WARN fallback last. Also exports `FAILURE_MODE_DEFAULTS` (the D-14 per-class default sets) and `inferToolCallSurface` (helper) so the generator can re-use them. Skips `~`-prefixed aliases (Pitfall 3) by emitting `null` for them (the generator filters).

2. **`scripts/capabilities/__fixtures__/openrouter-models-snapshot.json`** — frozen golden fixture with 5-10 representative entries covering: gpt-oss-120b (envelope leak anchor), gpt-oss-120b:free (variant), claude-3.5-sonnet (frontier), claude-3-haiku (mid-tier override), o1 (reasoning), deepseek-r1 (reasoning_tag_leak override), qwen-2.5-72b-instruct (open weight), gemini-2.0-flash-001 (frontier), meta-llama/llama-3.3-70b-instruct (open weight), `~anthropic/claude-sonnet-latest` (skip target). Hand-edited; offline test source of truth.

3. **`scripts/refresh-model-registry.mjs`** — CLI entrypoint. Default mode fetches OpenRouter `/api/v1/models`, transforms via `classify`, sorts by `(adapter, id)`, writes `packages/lattice/src/capabilities/registry.generated.ts` with the deterministic DO-NOT-EDIT header. `--check` mode regenerates to memory and diffs against the committed file; exits 1 on drift (D-17) and exits 0 with a stderr WARN on upstream fetch failure (D-18). Zero external runtime deps (node: built-ins only). 3-retry exponential backoff on fetch failures.

4. **`packages/lattice/test/capabilities-classifier.test.ts`** — vitest suite that exercises `classify` against the fixture and asserts the anchor cases (gpt-oss-120b -> open_weight_instruct + internal_envelope_leak, deepseek-r1 -> reasoning_tag_leak, claude-3-haiku -> mid_tier_rlhf, unknown prefix -> open_weight_instruct fallback + stderr WARN, `~latest` -> null). Includes a golden-snapshot test asserting the classifier's output on the full 10-entry fixture matches `toMatchSnapshot()`.

This plan does NOT run the generator against the live API and does NOT touch `registry.generated.ts` — Plan 04 owns that, after this plan ships the classifier + script that make it possible.

Purpose: the classifier is the brain of the registry; locking it down with golden tests against a hand-curated fixture is the single most important quality gate before Plan 04 produces ~337 real profiles.
Output: 4 new files. Generator and classifier together are <300 LOC total, zero deps, deterministic output, tested offline.

       ```javascript
       export function classify(rawEntry) {
         const id = rawEntry.id;
         if (id.startsWith("~")) return null;  // Pitfall 3: skip ~latest aliases

         const [prefix, ...rest] = id.split("/");
         const stripped = stripVariant(id);
         const afterPrefix = stripped.includes("/") ? stripped.split("/").slice(1).join("/") : stripped;

         const prefixRule = PROVIDER_PREFIX_RULES[prefix];
         let trainingClass = prefixRule?.trainingClass ?? FALLBACK.trainingClass;
         let originFamily  = prefixRule?.originFamily  ?? FALLBACK.originFamily;
         let reasoningSurface = "none";
         let extraFailureModes = [];

         for (const override of FAMILY_OVERRIDES) {
           if (afterPrefix.includes(override.match)) {
             if (override.trainingClass)    trainingClass = override.trainingClass;
             if (override.reasoningSurface) reasoningSurface = override.reasoningSurface;
             if (override.knownFailureModesAdd) extraFailureModes = override.knownFailureModesAdd;
             break;  // first hit wins (D-03)
           }
         }

         if (!prefixRule) {
           console.warn(
             `[classifier] WARN — unknown prefix '${prefix}' for id '${id}'. Defaulting to ${FALLBACK.trainingClass}. Consider adding to PROVIDER_PREFIX_RULES.`,
           );
         }

         const baseFailureModes = FAILURE_MODE_DEFAULTS[trainingClass] ?? [];
         // De-duplicate while preserving order: defaults first, then overrides not already present.
         const seen = new Set(baseFailureModes);
         const knownFailureModes = [...baseFailureModes];
         for (const m of extraFailureModes) {
           if (!seen.has(m)) { seen.add(m); knownFailureModes.push(m); }
         }

         return {
           originFamily,
           trainingClass,
           reasoningSurface,
           toolCallSurface: inferToolCallSurface(rawEntry),
           knownFailureModes,
           recommendedPromptStrategy: PROMPT_STRATEGY_BY_CLASS[trainingClass],
         };
       }
       ```

       ```javascript
         return "native_lenient";
       }
       ```

       ```javascript
       async function fetchWithRetry(url, attempts = 3) {
         let lastErr;
         for (let i = 0; i < attempts; i += 1) {
           const ctrl = new AbortController();
           const timeoutId = setTimeout(() => ctrl.abort(new Error("timeout")), FETCH_TIMEOUT_MS);
           try {
             const resp = await fetch(url, {
               headers: { Accept: "application/json" },
               signal: ctrl.signal,
             });
             if (!resp.ok) throw new Error(`HTTP ${resp.status}`);
             return await resp.json();
           } catch (err) {
             lastErr = err;
             if (i < attempts - 1) {
               const backoffMs = 500 * Math.pow(2, i);
               await new Promise((r) => setTimeout(r, backoffMs));
             }
           } finally {
             clearTimeout(timeoutId);
           }
         }
         throw lastErr;
       }
       ```
